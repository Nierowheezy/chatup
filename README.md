### Setup .env file 

- cd backend 
- check the .env file
- add the configs for each as seen below

```js
MONGODB_URI=...
PORT=5001
JWT_SECRET=...

CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...

NODE_ENV=development
```

### Build the app

- Run this command 

```shell
npm run build
```


### Start the app

- Run this command 


```shell
npm start
```


### To seed mock user data into the database

- cd into backend
- run this command 

```shell
npm run seed 
```
